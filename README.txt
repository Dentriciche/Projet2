Final-Project | Module 2 - Student Performance

Project Description
In this project, we will study a dataset of student. The data comes from a thesis "USING DATA MINING TO PREDICTSECONDARY SCHOOL STUDENT PERFORMANCE" made by P. Cortez and A. Silva for the University of Minho, Portugal.
In this project we will only use the data with the result of portuguese class.
The data have been downloaded from https://archive.ics.uci.edu/ml/datasets/student%2Bperformance
The link to the thesis : http://www3.dsi.uminho.pt/pcortez/student.pdf


Project Goal
The objective of this project is to find insight on the database on relation with the grades on portuguese.
The final objective being to identify the factors that can allow one to foresee excellent students or poor students
